autoScrollTo
autoScrollTo: boolean
类型 : boolean
可选的

当下拉框有已选项，单选时，打开下拉框会自动滚动到已选项；当为多选时，打开下拉框会自动滚动到已选项的第一项；默认打开为true

badge
badge: number
类型 : number
可选的

类型 : boolean

isRequired
isRequired: boolean
类型 : boolean
可选的

类型 : boolean

isShowSearchBox: boolean
