输入属性

buttonBlock
类型 : any

继承父宽度 - <np-button buttonBlock>继承父宽度</np-button>
buttonSize
类型 : "large" | "default" | "small"

按钮宽度大小 - large - 300px；default - 180px；small - 80px。
buttonType
类型 : "primary" | "secondary" | "cancel" | "search"

按钮类型

primary - 主色调按钮，即Pluto色；secondary - 副色调按钮；cancel - 取消按钮；search - 搜索按钮
file

二进制文件。

funcType
类型 : "normal" | "upload"
缺省值 : 'normal'

按钮功能类型 - normal - 默认类型；upload - 即为上传文件类型按钮。
isDisabled
类型 : boolean

是否禁用按钮。

uploadFileType
缺省值 : ''

上传文件类型， 如：[uploadFileType]="'jpg|png|jpeg'"。

特别说明一下格式，因为采用正则匹配，所以传入格式为'xlsx|xls'，用|隔开。

输出属性

fileChange
类型 : EventEmitter

文件改变触发事件， 即双向绑定。

uploadErrorMessage
类型 : EventEmitter

上传文件出错消息事件。
