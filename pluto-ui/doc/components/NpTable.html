输入属性

columns
类型 : Array<ColumnDef>

定义列名称（即数据源datasource对象属性名称），列显示名称，列宽度（可选，默认每列自适应等宽）

datasource
类型 : Array<any>

表格组件数据源，数据源为一个对象数组，对象的属性名称必须与表格列名称相同，否则不会展示此列

firstBtnText
类型 : string
缺省值 : '首页'

首页按钮文字

itemsPerPageText
类型 : string
缺省值 : '条，每页显示条数'

noDataLabel

pageNumber

pageSize

serverSideSort
类型 : boolean
缺省值 : false

是否服务器端排序，默认false，即客户端排序

showCheckbox
类型 : boolean
缺省值 : false

showPager
类型 : boolean

输出属性

onPageChanged
类型 : EventEmitter

页码改变事件，即翻页事件

onPageSizeChanged
类型 : EventEmitter

每页显示条数改变事件

onRows
类型 : EventEmitter

当前处理的行row/rows，选中/非选中状态

如点击全选按钮，则处理的是当前页的所有行rows
onRowsChecked
类型 : EventEmitter

Will be deprecated, use onRows instead
onRowsUnchecked
类型 : EventEmitter

Will be deprecated, use onRows instead

为了向后兼容，故没有修改onRowsChecked逻辑，增加此output事件

当前行uncheck时触发此事件
onServerSideSorted
类型 : EventEmitter

当选择服务器端排序（即serverSideSort = true）时，根据当前点击列内容及排序，传给backend api返回服务器端排序数据赋值给datasource及total字段即可

pageNumberChange
类型 : EventEmitter

页码改变事件，即翻页事件 （双向绑定）
